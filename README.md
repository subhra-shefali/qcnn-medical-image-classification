# Quantum Convolutional Neural Network (QCNN) for Medical Image Classification

## Overview
This project focuses on developing a Quantum Convolutional Neural Network (QCNN) for classifying medical images such as skin cancer, chest X-ray, and histopathology images. The aim is to explore the integration of quantum computing with deep learning techniques for medical image analysis.

## Technologies Used
- Python  
- PyTorch  
- PennyLane / Qiskit  
- NumPy  
- Matplotlib  
- Scikit-learn  

## Dataset
The model was trained and evaluated on medical imaging datasets, including:
- Skin Cancer Images  
- Chest X-ray Images  
- Histopathology Images  

## Project Workflow
- Data preprocessing and normalization  
- Feature encoding into quantum circuits  
- Implementation of QCNN architecture  
- Model training and optimization  
- Performance evaluation  

## Results
- Achieved classification performance using QCNN model  
- Evaluated results using accuracy, confusion matrix, and classification report  
- Demonstrated the feasibility of hybrid quantum-classical models in medical imaging tasks  

## Future Improvements
- Improve model accuracy with better hyperparameter tuning  
- Experiment with larger datasets  
- Optimize quantum circuit design for better performance
